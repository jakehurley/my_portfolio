This project looked at creating a tool for prospective and current AirBnB owners that helps price their listings and estimate revenue. This was executed using data from "Inside AirBnB" which had metrics on listing locations, prices, bedrooms and more. Using a Ridge regression model, price was predicted with an R2 score of ~0.60. An XBoosted decision tree was used to predict the revenue with a score of ~0.34.
